Folders & Structure

Keep your notes organized. Virtual folders are automatically synced to your Google Drive / ViNotes directory.

Instant Sharing

Share any note via a direct link or QR Code. Others can import your updates instantly.

Quick Commands

:share Creates a shareable link for the current note.
:qr Shows a QR Code for fast mobile access.
:file View file details and Drive sync status.

Tip: Type these inside the editor to trigger actions

Using ViNotes

How do these commands work?

Just type a command (like :share) on a new line. ViNotes will detect it, remove the text, and run the action instantly.

Where is my data stored?

By default, notes are local. When synced, they are stored in a folder called ViNotes in your Google Drive root.
